# Resume

This folder contains my resume in **Markdown** (`Resume.md`) and the generated **PDF** (`Resume.pdf`).

## How it works

- **Edit** `Resume.md`
- **Push** to `main`
- **GitHub Actions** generates a fresh `Resume.pdf` via the web converter at `https://nazarkulyk6.github.io/MDtoPDF/` and **commits the updated PDF** back to the repo (only if it changed).

## Files

- **`Resume.md`**: source of truth
- **`Resume.pdf`**: auto-generated output
